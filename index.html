À propos de moi

Développeur web passionné en reconversion

Après plusieurs années dans le commerce, j’ai décidé de me reconvertir dans le développement web, une passion que je cultive depuis longtemps.


Aujourd’hui, je maîtrise les langages du web moderne comme JavaScript, TypeScript, CSS, ainsi que des frameworks tels que React et Node.js. Je continue d’élargir mes compétences en me formant actuellement sur Java et Spring Boot.


Mon objectif : concevoir des applications performantes, accessibles et maintenables, tout en évoluant continuellement dans un environnement stimulant.

  • 1.

    Développement front-end : HTML, CSS, JavaScript, React, TypeScript

  • 2.

    Développement back-end : Node.js, Express, SQL

  • 3.

    Apprentissage en cours : Java & Spring Boot

  • 4.

    Méthodologie : projets pratiques, clean code, bonnes pratiques
